# International Yoga Festival 2026

Event Management System built with Laravel 12.

## Features

- Online Registration
- Participant Portal
- Email Notification
- QR Code Ticket
- QR Check-In
- Admin Dashboard
- Payment Verification
- Xendit Payment Gateway (In Progress)

## Tech Stack

- Laravel 12
- MySQL
- TailwindCSS
- Laravel Breeze
- Simple QR Code
- Xendit API
